2 The correction:

Zoom in on the formula. α scales the majority rank we compare against: α = 1 → full correction keff = r, α = 0 → no correction (rank 1). Empirically the optimal α is usually below 1 because minority clusters violate the homogeneous-Poisson assumption.

The blue line connects the query to the keff-th nearest majority.

4 Joint CV — pick (α*, n_votes*) from data

KNNFairRankJointCV does inner stratified 3-fold cross-validation over the full grid of 4 α × 6 n_votes = 24 pairs. For each pair, average G-Mean over 3 folds; commit to the argmax. Click "Run inner CV" to animate the 72 fold evaluations (24 cells × 3 folds), with the active fold-split shown below.

Five stages from training data to a prediction. The efficiency trick: α only affects prediction, not the neighbourhood structure — so we fit once per (fold, n_votes) and sweep all α values cheaply on the same fitted model.
